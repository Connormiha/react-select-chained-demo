Tested with Node.js 6+
### Install
```
npm install
```

### Usage
Run http server + watcher
```
npm start
```

Open in your browser
```
http://localhost:8080/  
http://localhost:8080/webpack-dev-server/ with live reload
```

Run unit test
```
npm test
```

Check code style by eslint
```
npm run lint
```

Check code coverage
```
npm run coverage
```
